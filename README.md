# homework-15

# Описание

Одностраничное приложение (SPA) для просмотра информации о персонажах, локациях и эпизодах мультсериала **Rick and Morty**.

## Технологии

- **Vite**
- **React**
- **TypeScript**
- **React Router v6**
- **Локальные JSON-данные**

## Функциональность

- Главная страница с навигацией
- Список персонажей с сортировкой по дате создания (через query-параметры)
- Детальная страница персонажа
- Список локаций + детальная страница
- Список эпизодов + детальная страница
- Навигация доступна на всех страницах
- Страница 404 при неверном маршруте
- Типизация данных через TypeScript
- Используются project references (`tsconfig.app.json`, `tsconfig.node.json`)

## Структура проекта

```bash
src/
  data/          # Локальные JSON-данные
  layouts/       # Основной layout с навигацией
  pages/         # Страницы приложения
  types.ts       # Типы данных TypeScript
  main.tsx       # Точка входа
```

## Установка и запуск

```bash
npm install
npm run dev
```
